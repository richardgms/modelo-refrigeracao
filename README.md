# Modelo Refrigeração

Este é um projeto de landing page para serviços de refrigeração e ar condicionado, construído com Vite.

## Pré-requisitos

Certifique-se de ter o Node.js instalado em sua máquina.

## Instalação

Para instalar as dependências do projeto, execute:

```bash
npm install
```

## Desenvolvimento

Para iniciar o servidor de desenvolvimento local:

```bash
npm run dev
```

O servidor geralmente estará disponível em `http://localhost:5173`.

## Build (Produção)

Para criar a versão de produção do projeto:

```bash
npm run build
```

Os arquivos otimizados serão gerados na pasta `dist`.

## Preview

Para visualizar a versão de produção localmente:

```bash
npm run preview
```

## Deploy

Para fazer o deploy, você pode usar serviços como Vercel, Netlify ou qualquer hospedagem estática. Geralmente, você precisará configurar o comando de build como `npm run build` e o diretório de saída como `dist`.
